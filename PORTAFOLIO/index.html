Proyectos Web

Actividades de la materia agrupadas por unidad. Cada proyecto es interactivo y se ejecuta desde este portafolio.

Unidad 1

Horario de clases en tabla HTML y CSS

Actividad donde se diseña un horario de clases utilizando una tabla de HTML con estilos en CSS para diferenciar las materias por colores, aplicar bordes colapsados y mejorar la legibilidad del contenido.

Ver en pestaña completa

Unidad 2

Biblioteca digital con filtros dinámicos

Biblioteca digital donde se muestran libros con filtros por autor, editorial y rango de precios. El contenido se actualiza de forma dinámica utilizando JavaScript y peticiones asíncronas a un servidor en PHP, sin recargar la página completa.

Ver en pestaña completa

Unidad 3

Juego tipo Pac-Man con JavaScript

Juego inspirado en Pac-Man desarrollado con HTML, CSS y JavaScript. Se utilizan eventos de teclado, manipulación del DOM y lógica de colisiones para mover al personaje por el mapa, recoger ítems y evitar a los enemigos.

Ver en pestaña completa

Unidad 4

Editor colaborativo multi-lenguaje P2P

Editor de código en línea que permite escribir y compartir programas en C, DOG y Chicken de forma colaborativa. Utiliza conexiones P2P para sincronizar el texto entre usuarios, muestra cursores remotos en tiempo real y cuenta con una consola donde se ejecuta el código y se ve la salida directamente en el navegador.
